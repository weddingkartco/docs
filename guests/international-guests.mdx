---
title: "International Guests"
description: Add and message guests outside India
---

Weddingkart supports guests in any country. The process for adding international guests is the same as for local guests — the only requirement is that you include the full international phone number with the country code.

## Adding international guests manually

<Steps>
  <Step title="Open the Guests section">
    Navigate to the wedding in the app and open **Guests**.
  </Step>
  <Step title="Tap Add Guest">
    Tap **Add Guest** → **Add Manually**.
  </Step>
  <Step title="Enter the international phone number">
    Enter the guest's name and phone number with the full country code.

    | Country | Example format |
    |---------|---------------|
    | United States | `+1 555 123 4567` |
    | United Kingdom | `+44 7911 123456` |
    | UAE | `+971 50 123 4567` |
    | India | `+91 98765 43210` |

    Include the `+` prefix and the country code before the local number.
  </Step>
  <Step title="Save">
    Tap **Save**. The guest is added with their international number.
  </Step>
</Steps>

## Adding international guests via Excel

For large groups of international guests, include their numbers in your Excel file with the country code in the phone column. Weddingkart reads the country code during import and stores the number in the correct format.

<Tip>
  Always verify international phone numbers before importing. A wrong country code means WhatsApp messages will not be delivered.
</Tip>

## How Weddingkart handles missing country codes

If a phone number in your guest list has no country code, Weddingkart assumes it belongs to the same country as your account.

For example: if your Weddingkart account is registered with an Indian number (`+91`), any guest number without a country code is assumed to be an Indian number.

This means:

- If all your guests are in the same country as you, you can safely omit country codes.
- If you have a mix of local and international guests, include the `+` country code for every international number to avoid incorrect assumptions.

<Warning>
  Do not mix local and international guests in the same import without country codes. Weddingkart will apply your account's country code to all numbers that lack one, which will cause delivery failures for your international guests.
</Warning>

## WhatsApp messaging for international guests

WhatsApp messaging works the same way for international guests — they receive messages in the WhatsApp app they already use, with no additional setup required on their end.

However, WhatsApp charges different rates per message depending on the destination country. Indian guests cost 1 credit per message. International guests may cost more credits depending on their country's WhatsApp rate.

Check the pricing details before sending bulk announcements to international guests.
